Issues of MMORPGS

Health

When playing MMORPG's sometimes users can go on for hours on end playing with no breaks or physical activity. This can lead towards several health defects and once the user finally decides to take action against these bad habits it might be too late.

Financial Issues

Some MMORPG's have monthly subscription based payments which can start to add up if the user doesn't have a career, can't play hardcore if you have a career.

Monthly subscriptions aren't the only issue when a game is "Free to Play" that doesn't mean that the user won't spend money on in game transactions , which can be very persuading when it can give the user a major advantage over others
